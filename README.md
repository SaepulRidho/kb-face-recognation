# KB Face Recognition Anime Similarity

## Description
KB Face Recognition adalah project berbasis Artificial Intelligence yang dapat mendeteksi tingkat kemiripan wajah pengguna dengan karakter anime menggunakan teknologi face recognition dan image similarity.

---

## Features
- Face Detection
- Anime Similarity Detection
- Percentage Similarity Result
- Real-Time Detection
- AI Recognition

## Technologies Used
- Python
- OpenCV
- NumPy
- Face Recognition
- Computer Vision

## How It Works
1. Sistem mengambil gambar wajah pengguna
2. Wajah dideteksi menggunakan AI
3. Sistem membandingkan dengan dataset anime
4. Hasil similarity ditampilkan dalam persentase

## Installation

```bash
git clone https://github.com/SaepulRidho/kb-face-recognation.git
cd kb-face-recognation
pip install -r requirements.txt
python app.py
```
